# EVA-KI-gestützter-Versandassistent
EVA analysiert Versanddaten, vergleicht Carrier-Tarife, bewertet Versicherungsanforderungen und empfiehlt die wirtschaftlichste Versandoption auf Basis hinterlegter Excel-Daten. Die regelbasierte Entscheidungslogik wird durch generative KI ergänzt, um Ergebnisse verständlich zu erklären und Nutzer bei Versandentscheidungen zu unterstützen.
